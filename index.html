Загальні налаштування безпеки в операційних системах

Облікові записи користувачів

  • Різні рівні доступу для адміністраторів і звичайних користувачів
  • Можливість створення, видалення та налаштування облікових записів з різними правами доступу

Політика паролів

  • Вимоги до складності паролів (мінімальна довжина, використання великих і малих літер, цифр і спеціальних символів)
  • Регулярна зміна паролів і заборона використання старих паролів
  • Відстеження спроб входу та блокування облікового запису після кількох невдалих спроб

Шифрування даних

  • Шифрування дисків і файлів для захисту від несанкціонованого доступу
  • Використання шифрування під час передавання даних у мережі (HTTPS, VPN тощо)

Антивірусне ПЗ

  • Інтеграція з антивірусними програмами для захисту від шкідливих програм
  • Регулярні оновлення антивірусних баз

Оновлення системи

  • Автоматичне завантаження та встановлення оновлень безпеки
  • Ручне налаштування та контроль за оновленнями

Брандмауер (Firewall)

  • Контроль вхідного та вихідного трафіку щоб запобігти несанкціонованому доступу
  • Налаштування правил брандмауера для різних типів мережевих з'єднань

Windows

операційна система, розроблена компанією Microsoft, є однією з найпоширеніших ОС для персональних комп'ютерів.

  • Можливість використання локальних облікових записів або облікових записів Microsoft для синхронізації налаштувань та доступу до хмарних сервісів.
  • Windows Hello: використання біометричних даних (відбитків пальців, розпізнавання обличчя) для входу в систему.
  • BitLocker: шифрування дисків для захисту даних від несанкціонованого доступу.
  • EFS (Encrypting File System): шифрування окремих файлів і папок.
  • Windows Defender Firewall: вбудований брандмауер для контролю мережевих з'єднань і захисту від загроз.

macOS

операційна система для комп'ютерів Mac, розроблена компанією Apple. Має високий рівень захисту від вірусів і шкідливих програм.

  • iCloud облікові записи: синхронізація налаштувань і даних через iCloud.
  • Touch ID та Face ID: використання біометричних даних для входу в систему та підтвердження покупок.
  • FileVault: шифрування всього диска для захисту даних.
  • Keychain: захищене зберігання паролів і сертифікатів.
  • Вбудований брандмауер macOS: контроль вхідного та вихідного трафіку для захисту системи.

iOS

операційна система розроблена компанією Apple, що використовується на смартфонах iPhone, планшетах iPad, розумних годинниках Apple Watch та інших пристроях Apple. Має високий рівень захисту даних користувачів та контроль доступу додатків. Високоякісний контроль додатків у App Store забезпечує надійність та безпеку програм.

  • Apple ID: синхронізація даних і налаштувань через обліковий запис Apple.
  • Можливість налаштування облікових записів для різних членів родини.
  • Використання паролів або PIN-кодів для розблокування пристрою.
  • Face ID та Touch ID: використання біометричних даних для розблокування пристрою та підтвердження покупок.
  • Шифрування всього диска для захисту даних.
  • iCloud Keychain: захищене зберігання паролів і сертифікатів.
  • Вбудовані функції для контролю мережевого трафіку та захисту даних.

  • Синхронізація даних і налаштувань через обліковий запис Google.
  • Використання паролів, PIN-кодів або малюнків для розблокування пристрою.

Оновлення ОС та іншого ПЗ на пристроях

Чому оновлення є критично важливими?

Виправлення вразливостей

Вразливості у ПЗ можуть бути виявлені як розробниками, так і зловмисниками. Останні використовують їх для атак на систему. Оновлення містять виправлення цих вразливостей, запобігаючи можливим загрозам.

Покращення функціональності

Оновлення ПЗ також додають нові функції, покращуючи загальну продуктивність та користувацький досвід.

Забезпечення сумісності

Регулярні оновлення допомагають зберігати сумісність ПЗ з новими технологіями та іншими програмами, що забезпечує безперебійну роботу системи.

Захист від шкідливого ПЗ

Оновлення містять нові бази даних загроз, що дозволяє антивірусному програмному забезпеченню виявляти та блокувати нові типи шкідливого ПЗ.

Як перевіряти та встановлювати оновлення на різних операційних системах?

Оновлення можуть бути доступні для завантаження вручну або автоматично через систему оновлень

Windows
  • Перейдіть до "Пуск > Налаштування > Оновлення та безпека > Центр оновлення Windows".
  • Натисніть перевірити наявність оновлень для ручної перевірки.
  • Або переконайтеся, що опція “Завантажувати та встановлювати оновлення автоматично” увімкнена.
macOS
  • Відкрийте "Системні налаштування > Оновлення ПЗ".
  • Натисніть "Перевірити наявність оновлень".
  • Або встановіть галочку поруч із "Автоматично перевіряти оновлення", "Завантажувати нові оновлення за наявності" та інші відповідні опції.
Android
  • Відкрийте "Налаштування > Система > Оновлення системи".
  • Натисніть "Перевірити наявність оновлень".
  • Або увімкніть опцію "Автоматичні оновлення" або "Завантажувати оновлення автоматично".
iOS
  • Відкрийте "Налаштування > Основні > Оновлення ПЗ".
  • Натисніть "Перевірити наявність оновлень".
  • Або натисніть "Автоматичні оновлення" та увімкніть опції "Завантаження оновлень iOS" та "Інсталяція оновлень iOS".

Безпечне встановлення додатків на пристрої

соціальні мережі – онлайн-платформи, які...

Перевірте, які дозволи запитує додаток. Будьте обережні з додатками, які вимагають доступу до великої кількості конфіденційних даних. Ознайомтеся з політикою конфіденційності додатку, щоб зрозуміти, як ваші дані будуть використовуватися та захищатися.

Регулярно оновлюйте додатки до останніх версій, оскільки розробники часто виправляють у них вразливості та покращують безпеку.

Використовуйте тільки офіційні магазини додатків (Google Play Store, Apple App Store, Microsoft Store, Mac App Store), оскільки вони проводять перевірки безпеки та якості додатків перед публікацією.

Читайте відгуки та оцінки інших користувачів, щоб дізнатися про досвід використання додатку. Низькі рейтинги та негативні відгуки можуть бути ознакою проблем з безпекою або функціональністю.

Як керувати дозволами для додатків на різних платформах?

Завдяки цьому ви зможете обмежити доступ додатків до конфіденційних даних

1.

Перед встановленням додатку перегляньте список дозволів, які він запитує, та переконайтеся, що вони відповідають функціональності додатку.

2.

Перейдіть до керування дозволами для додатків:

для пристроїв на ОС Windows

Перейдіть до "Налаштування > Конфіденційність". Тут ви можете керувати дозволами для доступу до конфіденційних даних (камера, мікрофон, місцезнаходження) для кожного додатку.

для пристроїв на macOS

Перейдіть до "Системні налаштування > Безпека та конфіденційність > Конфіденційність". Тут ви можете керувати дозволами для доступу до конфіденційних даних  для кожного додатку.

для пристроїв на iOS

Перейдіть до "Налаштування > Конфіденційність". Тут ви можете переглядати та змінювати дозволи для доступу до конфіденційних даних для кожного додатку.

для пристроїв на Android

Перейдіть до "Налаштування > Програми > Дозволи або Менеджер дозволів". Тут ви можете переглядати та змінювати дозволи для кожного встановленого додатку.

3.

Обмежте доступ додатків до конфіденційних даних, якщо це не є необхідним для їхнього функціонування.

Уникнення шкідливого ПЗ на пристроях

Шкідливе програмне забезпечення (malware) – це будь-яке програмне забезпечення, створене з метою завдання шкоди комп'ютеру, мережам або користувачам

До шкідливого ПЗ відносяться віруси, трояни, черв'яки, програми-вимагачі (ransomware), шпигунські програми (spyware) тощо

Потенційні ризики від шкідливого ПЗ

Втрати даних

Шкідливі програми можуть видаляти або пошкоджувати файли, що призводить до втрати важливої інформації

Крадіжка особистих даних

Програми-шпигуни можуть збирати особисту інформацію, як-от паролі, номери кредитних карток та інші конфіденційні дані

Фінансові втрати

Програми-вимагачі можуть заблокувати доступ до ваших даних або систем, вимагаючи викуп за їх розблокування

Компрометація системи

Шкідливі програми можуть використовувати ваш комп'ютер для атаки на інші системи або для поширення шкідливого ПЗ

Порушення конфіденційності

Шкідливе ПЗ може надавати зловмисникам доступ до вашої системи, що дозволяє їм переглядати та використовувати вашу інформацію

Профілактичні заходи, що допоможутьуникнути загроз, пов’язаних із шкідливим ПЗ

Оновлення програмного забезпечення

Автоматичні оновлення

Увімкніть автоматичні оновлення для вашої ОС та ПЗ. Це допоможе забезпечити своєчасне встановлення патчів безпеки.

Патч – це модифікація програмного коду, яка усуває вразливість. Це може бути зміна логіки програми, додавання перевірок безпеки або виправлення помилок коду.

Регулярні перевірки оновлень

Регулярно перевіряйте наявність оновлень для програм, які не підтримують автоматичне оновлення, і встановлюйте їх.

Перевірка покликань

Уникайте натискання на покликання у підозрілих повідомленнях, електронних листах або на вебсайтах.

Наведіть курсор на покликання, щоб побачити його повний URL. Якщо адреса виглядає підозріло або відрізняється від очікуваної, не натискайте на нього.

Сканування вкладень

Перед відкриттям вкладень скануйте їх за допомогою антивірусного програмного забезпечення.

Не відкривайте вкладення, якщо вони здаються підозрілими або ви не очікували отримати їх. Це стосується особливо файлів з розширеннями .exe, .bat, .scr, .zip, .rar.

Використання антивірусного ПЗ

Антивірусна програма

Антивірусне ПЗ (Norton, McAfee, Bitdefender тощо) сканує ваші файли та програми для виявлення шкідливого ПЗ. Воно використовує бази даних, що постійно оновлюються, для розпізнавання нових загроз. Антивірус ізолює вірус або інше шкідливе ПЗ, запобігаючи його поширенню або пошкодженню вашого пристрою.

  • Встановіть антивірусне ПЗ на свій комп'ютер або мобільний пристрій.
  • Проводьте регулярне сканування системи на наявність вірусів та шкідливого ПЗ.
  • Регулярно оновлюйте антивірусні бази даних, щоб забезпечити захист від нових загроз.

Безпечне використання Інтернету

Обережність на вебсайтах

Уникайте відвідування підозрілих або ненадійних вебсайтів.

Використовуйте HTTPS-з'єднання для передавання конфіденційних даних. Перевіряйте наявність значка замка в адресному рядку браузера.

Використання VPN

VPN (віртуальна приватна мережа) – технологія, що дозволяє створити захищене та зашифроване з'єднання через Інтернет, приховуючи ваше реальне місцезнаходження та IP-адресу.

Це особливо корисно під час підключення до публічних мереж Wi-Fi, здійснення онлайн-покупок або під час роботи з конфіденційною інформацією.

Приклади VPN-сервісів: NordVPN, ExpressVPN, CyberGhost.

Дотримуючись рекомендацій щодо оновлення програмного забезпечення, уникнення підозрілих покликань, обережностіз вкладеннями в електронній пошті та використання антивірусного програмного забезпечення, ви зможете ефективно захистити свій комп'ютер або мобільний пристрій від шкідливих програм.

Оновлення ОС та іншого ПЗ на пристроях

Чому оновлення є критично важливими?

Шифрування захищає дані під час їх передачі через Інтернет або інші мережі. Використання протоколів HTTPS і VPN допомагає забезпечити захищене з'єднання.

Шифрування даних на жорсткому диску або мобільному пристрої захищає інформацію у разі втрати або крадіжки пристрою. Наприклад, BitLocker на Windows або FileVault на macOS забезпечують шифрування дисків.

Деякі програми дозволяють шифрувати окремі файли або папки для додаткового захисту (наприклад, VeraCrypt).

Шифрування резервних копій даних гарантує, що навіть якщо резервні копії будуть викрадені, зловмисники не зможуть прочитати інформацію без відповідного ключа.

Чому оновлення є критично важливими?

Зберігайте резервні копії у кількох місцях, наприклад, на зовнішніх жорстких дисках, у хмарних сховищах і на фізичних носіях (наприклад, USB-накопичувачах).

Використовуйте програмне забезпечення для автоматизації резервного копіювання, щоб не забути про це важливе завдання.

Визначте графік регулярного резервного копіювання залежно від важливості даних та частоти їх оновлення (щодня, щотижня або щомісяця).

Періодично перевіряйте можливість відновлення даних з резервних копій, щоб переконатися, що процес працює належним чином і дані можна успішно відновити.
